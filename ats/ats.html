ATS External Integration Guide

Introduction
This reference guide provides general information about each Applicant Tracking System (ATS) and specific information about each API operation and supported object type. This document provides an overview of how external systems can integrate with our ATS. It includes methods for establishing connections, retrieving job listings, and submitting job applications through various integration points.
Objects
In the ATS, the term object refers to a specific type of record used within the system. Examples of these objects include Job, Contact, Placement etc. Each object represents a fundamental concept in ATS, helping to structure how staffing data is stored, managed, and processed according to the systemâ€™s rules and logic.
